# AutoLab Annotator

A web-based video annotation tool for building AutoLab, a benchmark for evaluating Vision-Language Models on scientific laboratory experiment videos.

Annotators watch segmented lab videos and provide step-by-step descriptions of the procedures they observe. These human annotations serve as ground-truth references for grading VLM outputs.
